Primer proyecto destacado

INAE: Prevención de deserción estudiantil

Como muestra de nuestra capacidad resolutiva, desarrollamos INAE, una plataforma integral diseñada para instituciones de educación superior.

Combina monitoreo emocional, organización académica y mentorías entre pares, permitiendo a las Direcciones de Asuntos Estudiantiles (DAE) detectar alertas de riesgo en tiempo real y actuar de forma preventiva.

  • ✓ Dashboard analítico para administradores
  • ✓ Gestión segura y confidencial de datos
  • ✓ Sistema automatizado de alertas de riesgo
